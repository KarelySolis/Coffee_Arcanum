# Arcanum Coffee — Monorepo

Proyecto completo de Arcanum Coffee: API RESTful con FastAPI + PostgreSQL y UI con Next.js 14.

```
arcanum-coffee/
├── api/    → Backend FastAPI (puerto 8000)
└── web/    → Frontend Next.js (puerto 3000)
```

## Inicio rápido

1. Levantar el API → ver [api/README.md](api/README.md)
2. Levantar el Web → ver [web/README.md](web/README.md)
